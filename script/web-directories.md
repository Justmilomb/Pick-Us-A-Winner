# Web Directory Submissions — Backlink Strategy

## Why This Matters

Web directory backlinks:
1. Signal to Google that your site is real and established
2. Provide authoritative links that accelerate indexing
3. Bring direct referral traffic
4. Help with local/niche SEO

**Target:** 20-30 quality backlinks in first month → accelerates indexing from weeks to days

## Important: Quality > Quantity

**Avoid:**
- Spam directories (DA < 20)
- Link farms (selling links in bulk)
- Reciprocal link schemes ("I'll link to you if you link to me")
- PBN networks (private blog networks)

Google penalises low-quality backlinks. 10 links from DA 70+ sites are worth more than 100 links from DA 10 sites.
